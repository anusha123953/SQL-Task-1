# Library Management Database

## Description
Library Management Database project with SQL schema, ER diagram, and query implementation.

## Tables
- Users
- Books
- Borrow

## Relationships
- One User can borrow many Books
- One Book can be borrowed many times

## Features
- Primary Key
- Foreign Key
- One-to-Many Relationships

## Queries
- SELECT
- JOIN
- WHERE

## Tools Used
- MySQL
- draw.io
